# prograMEME-lang

# 프로그래밈 언어

# 문법

##출력 => {값} 물어본사람
정수일 경우 => 숫자
문자열인 경우 => 문자
변수인 경우 => 생략
```
숫자 3 물어본사람
문자 HelloWorld! 물어본사람
티비 물어본사람
```

##연산 => 곱해, 나눠, 더해, 빼
```
숫자 3 + 4
숫자 5 * 2
```

##변수 설정,변경 = 어쩔 {변수 이름} 는(은) {값}
변수 이름은 한글 쌉가능
변수에 정수를 연산할때는 {변수 + 정수} 순이어야 함
```
어쩔 세탁기는 문자 hello
어쩔 핸드폰은 숫자 5
어쩔 티비는 핸드폰
어쩔 티비는 세탁기 더하기 숫자 10
```

##입력 => 안물 {변수 이름}
변수 없을 경우 새로 만들어줌
정수로 받고 싶을 경우 뒤에 숫자 붙임
```
안물 티비
안물 핸드폰 숫자
```

##일반 파이썬 스크립트 => ㄴ(스크립트)ㄱ
한줄씩만 가능
```
ㄴprint("MEME")ㄱ
```
